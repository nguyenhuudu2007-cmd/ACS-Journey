# 🚀 ACS Journey: Lộ trình Kỹ sư Hệ thống Tính toán Nâng cao

> "Động lực giúp bạn bắt đầu, nhưng kỷ luật mới đưa bạn đến đích."

Chào mừng đến với không gian kỹ thuật số của tôi! Đây là repository lưu trữ toàn bộ hành trình 4 năm tự học và phát triển để trở thành một **Advanced Embedded & Edge Computing Engineer**. 

Tôi là một sinh viên năm nhất ngành Kỹ thuật Máy tính (Computer Engineering - EngCom) tại Đại học Bách Khoa TP.HCM (HCMUT). Repository này là minh chứng cho quá trình rèn luyện từ những dòng code C++ đầu tiên cho đến khi tự tay build một hệ điều hành Linux nhúng.

## 🎯 Mục tiêu cốt lõi
1. **Low-level:** Nắm vững quản lý bộ nhớ C/C++ và thao tác trực tiếp với thanh ghi (Register/Bare-metal).
2. **Mid-level:** Làm chủ RTOS, xử lý đa luồng, và kiến trúc IoT phân tán (ESP32, AWS IoT).
3. **High-level:** Triển khai Embedded Linux, viết Device Driver và nhúng AI (OpenCV) xuống vi điều khiển biên (Edge Computing).

## 🛠 Tech Stack & Hardware
* **Ngôn ngữ:** C/C++, Python (cho AI/Scripting).
* **Vi điều khiển/SBC:** STM32 (F1/F4), ESP32 Dual-core, Orange Pi / Raspberry Pi 4.
* **Hệ điều hành:** FreeRTOS, Embedded Linux (Yocto/Buildroot).
* **Workspace:** Lenovo IdeaPad (Core i7 11th Gen, 12GB RAM).

## 📂 Cấu trúc Repository

Lộ trình này được chia thành 9 chặng rõ ràng, bám sát cả chương trình học trên trường và các dự án thực chiến:

* `01_FOUNDATION/` - Nền tảng OOP, Memory Management C++ và Cấu trúc dữ liệu.
* `02_STM32_HAL/` - Vỡ lòng phần cứng STM32 bằng thư viện HAL (GPIO, Timer, ADC, UART, I2C).
* `03_STM32_REGISTER/` - Đào sâu vào lõi kiến trúc ARM Cortex-M, cấu hình thanh ghi.
* `04_STM32_BAREMETAL/` - Viết driver từ con số 0. Project: Hệ thống an ninh mini.
* `05_RTOS_DMA/` - Xử lý đa nhiệm với FreeRTOS và DMA. Project: Data Logger nâng cao.
* `06_ESP32/` - Giao thức mạng và IoT. Project: Gateway IoT 2 tầng (MQTT).
* `07_OPENCV/` - Tích hợp Computer Vision cơ bản.
* `08_LINUX_EMBEDDED/` - Khởi tạo môi trường Linux trên Orange Pi/Raspberry Pi.
* `09_CAPSTONE/` - Trùm cuối: Tích hợp toàn bộ hệ thống (AI Edge Node điều khiển STM32/ESP32).
* `PROGRESS_JOURNAL/` - Nhật ký tiến độ (Daily/Weekly log) và Bug Tracker.
